# Recipe Manager
Приложение для управления рецептами и списком покупок.

# Установка
```bash
pip install -r requirements.txt
```
# Запуск тестов
```bash
pytest test_recipes.py
```
# Структура
- `recipes.py` — классы Ingredient, Recipe, ShoppingList, DietaryRecipe
- `test_recipes.py` — тесты для класса Ingredient
